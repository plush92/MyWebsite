JavaScript Object Notation

How to use the JSON data:
Parse the data using JSON.parse()
Example:
    const jsonString = '{"name": "John", "age": 30}'; //original JSON data
    const obj = JSON.parse(jsonString); //Putting the data into a variable so we can apply methods & such
    console.log(obj.name); // Output: John //console log the JSON data and the 'name' key

    Additional: convert the data into a string using "stringify"
        const obj = { name: "John", age: 30 };
        const jsonString = JSON.stringify(obj);
        console.log(jsonString); // Output: '{"name":"John","age":30}'

Usuaully when you get the data it's in the form of a 'flat pack' string
