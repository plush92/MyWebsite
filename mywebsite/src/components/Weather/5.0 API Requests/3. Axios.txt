Axios - JavaScript library for API requests
Installation:
    npm: $ npm install axios
Import:
    ES6: import axios from 'axios';
    Common JS: const axios = require('axios');
    CDN: <script src="https://cdn.jsdelivr.net/npm/axios@1.6.7/dist/axios.min.js"></script>
Request methods:
    axios.get(url[, config])
    axios.post(url[, data[, config]])
    axios.put(url[, data[, config]])
    axios.delete(url[, config])
    **Look up Async/Await**

Response Schema
    The Axios response object contains:
        data: The server's response data.
        status: HTTP status code (e.g., 200).
        statusText: HTTP status message (e.g., 'OK').
        headers: HTTP headers from the server (access with response.headers['header-name']).
        config: The Axios config used for the request.
        request: The request object (ClientRequest in Node.js or XMLHttpRequest in the browser).

    Error Structure
    Axios errors contain:
        message: Summary of the error.
        name: Error type (always 'AxiosError').
        stack: Stack trace.
        config: Axios config used for the request.
        code: Axios-specific error code.
        status: HTTP status code.
